# HorseInACoat

Приложение для генерации случайных пользователей через API RandomUser.me с поддержкой оффлайн-режима и кешированием.

## Архитектура

**Clean Architecture + MVVM** с использованием современных Android технологий:

- **Kotlin** + **Jetpack Compose**
- **Clean Architecture** (Data - Domain - Presentation)
- **Hilt** для Dependency Injection
- **Room** для локальной базы данных
- **Retrofit** для API запросов
- **Coroutines** + **Flow**
- **Jetpack Compose** + **Material 3**
- **Navigation Compose**
- **Coil** (Image Loading)

- Получение случайных пользователей из API
- Фильтрация по полу и национальности
